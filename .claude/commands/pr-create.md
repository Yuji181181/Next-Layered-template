# Phase 4: PR & Remote Review Loop

プルリクエストを作成し、レビューを受けるためのワークフローです。

1.  **PR作成**: `gh pr create` を使用してプルリクエストを作成する。
    - *Note: GitHub側で自動的にCopilot Reviewerが起動します。*

2.  **待機 (Wait 10 mins)**:
    - **重要**: Copilotのレビュー完了には時間がかかります。
    - PR作成後、次のアクション（レビュー取得）を起こす前に、**必ず10分間の待機時間を設けてください**。
    - （`sleep 600` コマンドを実行するか、ユーザーに10分待つよう伝えて待機する）

3.  **レビュー取得**: `gh` コマンド等を使用して、PRについたレビューコメントを取得する。

4.  **妥当性検証 (Agent as a Judge)**:
    - `review-judgment` スキルに基づき、指摘の妥当性を検証する。
    - 不当な指摘であれば修正をスキップする。

5.  **修正**:
    - 必要な修正を行い、追加コミットをプッシュする。
